# 微信小程序-微信端源代码
 
# 使用说明
#### 下载源码 用微信开发工具打开  在 **utils-api.js** 替换自己的域名相关信息即可。
#### 后台代码:[https://gitee.com/xshuai/xai](https://gitee.com/xshuai/xai)
### 2019年12月23日更新说明

1.showLoading在wx.chooseImage中执行不显示问题修复，并没有全量提交代码。可以参考如下代码进行修改:

```
setTimeout(function () {
  wx.showLoading({
	title: "智能提取中...",
	mask: true
  })
}, 500);
```
2.如果微信版本更新为7.0.9貌似这个BUG 已经被官方修复
